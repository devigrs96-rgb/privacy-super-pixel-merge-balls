<h1>Política de Privacidade - Super Pixel Merge Balls</h1>
<p><strong>Última atualização:</strong> 1 de novembro de 2025</p>

<p>O jogo <b>Super Pixel Merge Balls</b> foi desenvolvido por Igor Gonçalves Ribeiro Silva como um aplicativo gratuito de entretenimento.</p>

<h2>Coleta e uso de informações</h2>
<p>O jogo não coleta nem armazena informações pessoais identificáveis. No entanto, utiliza serviços de terceiros, como <a href="https://unity.com/legal/privacy-policy" target="_blank">Unity Ads</a>, que podem coletar dados anônimos para exibir anúncios.</p>

<h2>Armazenamento local</h2>
<p>O jogo salva localmente informações como pontuação, sem enviar dados a servidores externos.</p>

<h2>Crianças</h2>
<p>O jogo é destinado a todos os públicos e não coleta dados de crianças intencionalmente.</p>

<h2>Alterações</h2>
<p>Podemos atualizar esta Política periodicamente. Recomendamos verificar esta página de tempos em tempos.</p>
